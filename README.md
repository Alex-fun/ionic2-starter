# ionic2-starter
An Ionic2 starter project

<img src="https://dn-cnode.qbox.me/FpA5jex9jPVAHpTIUqVbMQdXEvmd" width="260" />

#### start
```
git clone https://github.com/Alex-fun/ionic2-starter
cd ionic2-starter
npm i
```

#### run
```
ionic serve
```
